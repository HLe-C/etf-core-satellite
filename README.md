# ETF 核心-卫星策略

面向普通家庭的 A 股 ETF 核心-卫星组合构建、回测与执行规则研究。本项目适配“人工智能与投资研究”大作业 B2 题目，最终版本强调普通家庭可执行性、较低回撤、清晰调仓规则和可复现实证结果。

## 最终版本：V2.9 家庭配置策略

V2.9 不再把全部资金放入高波动风险策略，而是采用“风险策略 + 黄金 + 短债”的家庭组合框架：

- **风险策略 60%**：核心-卫星 ETF 动量策略。
- **黄金 ETF 20%**：`518880`，作为危机和通胀环境下的防守资产，但设置 20% 上限，避免过度依赖黄金样本期强势表现。
- **现金/短债 20%**：`511360` 短融ETF，作为低波动防守仓。
- **调仓频率**：每月第一个交易日检查并调仓。
- **数据约束**：所有信号只使用调仓日及之前的数据，避免未来函数。
- **交易成本**：按单边万分之一计入。

## 策略规则

### 外层家庭组合

1. 60% 配置到风险策略。
2. 20% 配置到黄金 ETF：`518880`。
3. 20% 配置到短债/现金仓：`511360` 短融ETF。
4. 黄金仓固定上限 20%，短债/现金仓不低于 20%。

### 风险策略内部

1. **核心池**：`510300` 沪深300、`510500` 中证500、`512890` 红利低波、`513100` 纳指。
2. **卫星池**：行业/主题 ETF 池中剔除核心标的和黄金。
3. **市场状态**：用沪深300 MA50/MA200 判断 bull、range、bear。
4. **卫星筛选**：收盘价高于 MA200，20 日动量为正，再按 60 日动量排名取前 2。
5. **风控规则**：MA20 止损、沪深300 单日大跌熔断、ATR 止盈、仓位偏离再平衡。
6. **调仓纪律**：月度执行，避免频繁交易和普通家庭难以跟随的问题。

## 研究结论

V2.9 是当前项目的最终主线版本。它牺牲了一部分极端上涨年份的进攻性，换取更低最大回撤、更稳定的年度表现和更接近普通家庭可执行的调仓负担。后续若继续研究，重点不应简单提高权益仓位，而应围绕防守资产替代、黄金失效情景、税费滑点和真实执行约束做稳健性验证。
